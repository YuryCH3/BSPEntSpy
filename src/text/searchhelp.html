Search HELP

Find Next

Finds and selects the next matching entity relative to the start of selection or selected entity. Hold Shift to add the matched entity to a selection.



Find all

Finds and selects all the matching entities. Hold Shift to add all the matched entities to a selection.

Search syntax HELP

The entities can be found by inputting a search-string into the search bar. By default the entity is matched if and only if a classname or targetname is equal to or contains a search-string.


There is an option to use advanced search, this way it is possible to precisely specify the searched key-values of an entity.

Example:

@"targetname" = "zombie1" @"squadname" = "ravenholm_zombies"

This search-string is going to match only entities who's targetname is "zombie1" AND squadname parameter is "ravenholm zombies".
